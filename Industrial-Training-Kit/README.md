# 🍽️ Saffron Table - Responsive Restaurant Website

A modern, fully responsive restaurant website built using **HTML5, CSS3, and Vanilla JavaScript**. This project demonstrates responsive frontend development, semantic HTML, modern UI/UX design, and interactive web components without using any frontend frameworks.

## 📌 Project Overview

Saffron Table is a premium restaurant landing page designed to provide an elegant and user-friendly experience across all devices. The project follows a mobile-first approach and showcases modern frontend development practices.

## ✨ Features

- Fully Responsive Design
- Mobile-First Layout
- Sticky Navigation Bar
- Hero Section
- About Section
- Featured Dishes
- Interactive Menu
- Chef Section
- Customer Testimonials
- Image Gallery
- Reservation Form with Validation
- FAQ Accordion
- Contact Section
- Professional Footer
- Smooth Scrolling & Animations
- SEO-Friendly Structure
- Accessible Semantic HTML

## 🛠️ Technologies Used
